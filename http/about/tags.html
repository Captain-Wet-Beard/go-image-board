Standard Tags

Tags are pieces of information that can be associated with an image or collection that makes searching for the image/collection easier. Tags should be short and concise. Consider adding tags for the image's genre, theme, media, author, and important elements contained within the image.

Collections

Collections are tagged automatically by their member images. When an image is added or removed from a collection or when an image in a collection is tagged or untagged, the same tag operations are performed on a collection. Collections cannot be directly tagged.

MetaTags

These are special tags that are automatically associated with an image. These are built into Go! Imageboard, and not uploaded by users.

MetaTags follow the same general format. [TagName]:[comparator][value]. comparator is defaulted to "=" if not provided. Example, rating:everyone is converted to rating:=everyone in the background. Not all tags support the same comparators.

Name Usage Description Valid Comparators Valid Queries Example
Uploader uploader:[uploaderName] Returns only images which were uploaded by [uploadername] = Images, Collections uploader:johnsmith
Rating rating:[someRating] Returns only images which are rated [someRating]. "unrated" is the default rating for all images. = Images rating:everyone
Score score:[someScore] Returns only images which are scored [someScore]. 0 is the default score for all images. =, >, <, >=, <=, Images score:>5
AverageScore AverageScore:[someScore] Same as score:[someScore] =, >, <, >=, <=, Images averagescore:>5
TotalScore totalscore:[someScore] Returns only images which have a total score of [someScore]. 0 is the default score for all images. =, >, <, >=, <=, Images totalscore:<10
ScoreVoters scorevoters:[voters] Returns only images which have had [voters] amount of users vote on it. 0 is the default for all images. =, >, <, >=, <=, Images scorevoters:<=10
InCollection InCollection:[y/n] Returns only images are [y] or are not [n] in any collection. = Images InCollection:n
TagCount TagCount:[number] Returns only images are [y] or are not [n] in any collection. =, >, <, >=, <=, Images TagCount:<5
Name Name:[SomeName] Returns only images or collections that have a name like [name]. There are some special escaping rules with this tag.

To search for an underscore, search -_
