Standard Tags

Tags are pieces of information that can be associated with an image that makes searching for the image easier. Tags should be short and concise. Consider adding tags for the image's genre, theme, media, author, and important elements contained within the image.

MetaTags

Meta tags are special tags that are automatically associated with an image. These are built into Go! Imageboard, and not uploaded by users.

Name Usage Description Example
Uploader uploader:[uploaderName] Returns only images which were uploaded by [uploadername] uploader:johnsmith
Rating rating:[someRating] Returns only images which are rated [someRating]. "unrated" is the default rating for all images. rating:everyone
