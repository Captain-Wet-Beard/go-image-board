Standard Tags

Tags are pieces of information that can be associated with an image that makes searching for the image easier. Tags should be short and concise. Consider adding tags for the image's genre, theme, media, author, and important elements contained within the image.

MetaTags

These are special tags that are automatically associated with an image. These are built into Go! Imageboard, and not uploaded by users.

MetaTags follow the same general format. [TagName]:[comparator][value]. comparator is defaulted to "=" if not provided. Example, rating:everyone is converted to rating:=everyone in the background. Not all tags support the same comparators.

Name Usage Description Valid Comparators Example
Uploader uploader:[uploaderName] Returns only images which were uploaded by [uploadername] = uploader:johnsmith
Rating rating:[someRating] Returns only images which are rated [someRating]. "unrated" is the default rating for all images. = rating:everyone
Score score:[someScore] Returns only images which are scored [someScore]. 0 is the default score for all images. =, >, <, >=, <=, score:>5
AverageScore AverageScore:[someScore] Same as score:[someScore] =, >, <, >=, <=, averagescore:>5
TotalScore totalscore:[someScore] Returns only images which have a total score of [someScore]. 0 is the default score for all images. =, >, <, >=, <=, totalscore:<10
ScoreVoters scorevoters:[voters] Returns only images which have had [voters] amount of users vote on it. 0 is the default for all images. =, >, <, >=, <=, scorevoters:<=10
